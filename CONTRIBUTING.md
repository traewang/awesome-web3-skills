# Contributing to Awesome Web3 Skills

Thank you for your interest in contributing! This list curates Web3 AI Agent Skills — plug-and-play skills that give AI agents native access to crypto.

## Guidelines

- **One resource per pull request.** This makes it easier to review and discuss.
- **Use the following format:**
  - For table entries: `| Name | [repo-name](url) | Brief description. |`
  - For list entries: `- [org/repo-name](url) - Brief description.`
- **Add new entries at the bottom** of the relevant category.
- **Check for duplicates** before submitting.
- **Verify that the link works** and the repository is actively maintained.

## What to Submit

We accept:
- Official exchange AI Agent Skills hubs (e.g., Binance, OKX, Bitget)
- Blockchain/protocol-specific AI agent skills (e.g., Solana, Uniswap)
- Trading and analytics agent skills
- Skills aggregator platforms and directories
- Frameworks and platforms powering the skills ecosystem

## Quality Criteria

- **Official or well-maintained** - From a recognized exchange/protocol or with meaningful community adoption.
- **Follows SKILL.md format** or provides MCP server / agent integration.
- **Actively maintained** - Recent commits within the past 6 months.
- **Well-documented** - Has clear README and usage instructions.

## How to Submit

1. Fork this repository.
2. Create a new branch: `git checkout -b add-skill-name`.
3. Add your resource to the appropriate section in `README.md`.
4. Commit your changes: `git commit -m "Add [skill/platform name]"`.
5. Push to your fork: `git push origin add-skill-name`.
6. Open a Pull Request with a brief explanation of why this resource should be included.

## Suggesting New Categories

If you believe a new category is needed, please open an issue first to discuss it.
